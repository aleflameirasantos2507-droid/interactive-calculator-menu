# Interactive Calculator Menu

A beginner-friendly Python project that allows users to perform different operations on two numbers through an interactive menu system.

## Features

- Add two numbers
- Multiply two numbers
- Compare numbers and determine the greater value
- Enter new numbers without restarting the program
- Exit the application through a menu option
- Continuous interaction using loops

## Technologies Used

- Python 3

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/your-username/interactive-calculator-menu.git
```

2. Navigate to the project folder:

```bash
cd interactive-calculator-menu
```

3. Run the script:

```bash
python main.py
```

## Example

### Initial Menu

```text
Enter a value: 10
Enter another value: 5

Your values are 10 and 5

[1] Add
[2] Multiply
[3] Greater Number
[4] New Numbers
[5] Exit Program
```

### Example Output

```text
The sum of 10 and 5 is 15
```

```text
END OF PROGRAM
```
